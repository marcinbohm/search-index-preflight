# v0.0.1-prealpha release notes

SearchIndexPreflight is an experimental offline-first CLI for reviewing Elasticsearch/OpenSearch mappings, templates, and schema diffs before they reach production.

This is a source-only pre-alpha release intended for evaluation and feedback. It is not production-ready as a required CI gate.

## Included

- Static lint rules: `SIL001`, `SIL002`, `SIL003`
- Experimental diff rules: `DIF001`, `DIF002`, `DIF003`
- `lint`
- `diff`
- `rules list`
- `explain <RULE_ID>`
- console and JSON output
- fixtures and examples

## Try it from source

```bash
go test ./...
go run ./cmd/search-index-preflight diff \
  --base examples/field-type-change/base \
  --current examples/field-type-change/current
```

This example exits non-zero because `DIF001` is an error finding.

The `version` command reports `0.0.1-prealpha` for this release.

## Not included

- Release binaries
- Production-ready CI gate guarantees
- Config loading
- Suppressions
- Baseline mode
- YAML input
- SARIF/Markdown output
- GitHub Action release integration
- Cluster doctor
- Migration/versioning implementation
